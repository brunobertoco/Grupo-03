Gestão de Eventos Esportivos

Sobre o Projeto
O projeto Gestão de Eventos Esportivos consiste no desenvolvimento de uma plataforma web para centralizar a divulgação e gestão de passeios de bike e corridas de rua, conectando organizadores locais a atletas. 
O escopo funcional do sistema abrange três frentes principais: uma área pública para visualização da agenda de eventos e portfólio de fotos/vídeos; um módulo do atleta para cadastro, realização de inscrições em competições e configuração de alertas de novos eventos; e um painel do organizador para gerenciamento dos eventos e upload de mídias.

O projeto está sendo desenvolvido como Trabalho Final da disciplina de Processo de Desenvolvimento de Software (PDS) do curso de Sistemas de Informação da Universidade Federal de Uberlândia (UFU).

Funcionalidades Principais
- Acesso Público: Visitantes podem visualizar a agenda de eventos futuros e galerias de fotos de edições passadas sem necessidade de login.
- Gestão de Inscrições: Atletas cadastrados podem se inscrever em passeios e corridas.
- Notificações: Alertas configuráveis para novos eventos na região do usuário.
- Galeria Esportiva: Organizadores podem fazer upload de fotos e vídeos para criar um portfólio visual dos eventos.

Tecnologias Utilizadas
- Frontend: React.js
- Backend: Java com Spring Boot
- Banco de Dados: PostgreSQL
- Comunicação: API RESTful.
